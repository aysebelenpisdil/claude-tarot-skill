# Your Tarot Books

Place the PDF files of your own tarot books in this directory. Any number of
books is supported.

```
references/books/
├── your-first-book.pdf
├── your-second-book.pdf
└── extracted/        (generated by /tarot-setup, do not edit by hand)
```

Everything in this directory except this README is ignored by git and will
never be committed. Do not share the `extracted/` content either — it is
derived from copyrighted books and is for your personal use only.

After adding your PDFs, run `/tarot-setup` inside Claude Code to build the
per-card knowledge base. Readings are refused until extraction is complete.
